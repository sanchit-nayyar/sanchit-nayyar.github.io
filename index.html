  1. Python Script for Sorting
    2. It happens more often than not that we end up with a cluttered downloads folder, with hundreds of unsorted files and it becomes very difficult to manage them. It also slows down the process of loading in this case. To tackle such issues, this script can be handy.   2. Python Script for Naming
    3. Sometimes, the files in a particular directory are named in weird fashions. These cases include the times when we import photos from our smartphone, or put multiple downloaded files in the destination directory without following a naming convention. Managing these files can become a headache, and this script aims to resolve all such issues. It is capable of naming all files with a particular extension in the said directory in the form of <prefix>_<i>.<ext> which enables easy handling of files. The value of i in this convention varies from 1 to whatever the need is, and the other variable terms are input by the user.
    GitHub Repository for said Project
  3. Python Script YouTube Downloader
    4. All the YouTube Downloaders available on the Internet as on date are able to fetch the video in the required quality and all other requirements, but the only issue is that they need the video URL. This YouTube Downloader is capable of carrying out a search on YouTube and downloading the first video that comes as a response to the said search query. Hence, saving time which is one of the major purposes of automation.
    Future prospects for this script include being able to fetch the video name and asking the user if this is the right video before executing the download process.
